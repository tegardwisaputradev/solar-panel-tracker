# Solar Panel Tracker

Proyek miniatur panel surya dengan bantuan Arduino Nano untuk mengatur orientasi panel surya agar terkena cahaya matahari dengan efektif. Serta listrik DC yang dihasilkan bisa diubah menjadi listrik AC dengan inverter sederhana yang dapat menghidupkan bohlam rumah

# Barang Yang Digunakan
* Arduino Nano
* Servo Motor
* 2 LDR
* Panel Surya Mini
* TP4056
* IC NE555
* IC CD4049
* 2 MOSFET IRLZ44N
* TRAFO CT
* Dll
